## Лабораторная работа 1: Система управления курсами и преподавателями
### CourseLib

Отдельные классы и интерфейсы, используемые в системе.

#### Модули:
- `Models` - абстрактные и дочерние от них классы, используемые в системе.
- `Builders` - класс поэтапной сборки курса (для соблюдения паттерна Builder).
- `Services` - класс с описанием работы самого сервиса (с соблюдением паттерна Singleton).
