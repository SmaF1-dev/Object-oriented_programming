### CourseApp

Тестирование отдельных классов и интерфейсов, используемых в системе управления курсами.

#### Модули:
- `CourseBuilderTests.cs.cs` - класс с тестами для проверки сборщика курса.

- `CourseModelsTests.cs` - класс с тестами для классов, используемых в системе.
