## Лабораторная работа 1: Система управления курсами и преподавателями
### CourseApp

Консольное приложение для запуска системы работы с курсами. Оно затягивает класс системы из `CourseLib` и запускает его как консольное приложение.

#### Для запуска:
Собираем само приложение:
```bash
dotnet build
```
Запускаем:
```bash
dotnet run
```

Или

```bash
dotnet run --project CourseApp
```
