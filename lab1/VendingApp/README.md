## Лабораторная работа C#
### VendingApp

Консольное приложение для запуска вендингового аппарата. Оно затягивает класс вендингового аппарата из `VendingLib` и запускает его как консольное приложение.

#### Для запуска:
Собираем само приложение:
```bash
dotnet build
```
Запускаем:
```bash
dotnet run
```
