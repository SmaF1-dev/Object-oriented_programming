## Лабораторная работа 3: Управление заказами в службе доставки
### DeliveryApp

Пример работы с DeliveryLib в виде небольшой программы.

#### Для запуска:
Собираем само приложение:
```bash
dotnet build
```
Запускаем:
```bash
dotnet run
```

Или

```bash
dotnet run --project DeliveryApp
```
