## Лабораторная работа 3: Управление заказами в службе доставки
### DeliveryLib

Система работы с курсами в виде библиотеки

#### Модули:
- `Models` - абстрактные и дочерние от них классы, используемые в системе.
- `Builders` - класс поэтапной сборки заказа (для соблюдения паттерна Builder).
- `Services` - класс с описанием работы самого сервиса (с соблюдением паттерна Singleton).
- `Factories` - класс с фабрикой для выбора способа сборки заказа в зависимости от типа (с соблюдением паттерна Factory Method)
- `Observer` - классы для имитации пользователей, которые получают уведомление при изменении статуса заказа (с соблюдением паттерна Observer)
- `Pricing` - классы для вычисления стоимости заказа с учетом возможных акций и т.п. (с соблюдением паттерна Strategy)
- `State` - классы для изменения статуса заказа (с соблюдением паттерна State)
